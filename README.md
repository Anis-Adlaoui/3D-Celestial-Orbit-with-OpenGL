# 🌍 3D Celestial Orbit with OpenGL

A real-time 3D celestial visualization developed in C++ using  OpenGL.

The project renders a textured Earth surrounded by a dynamic star field while providing interactive camera controls and real-time animation. It was developed as a computer graphics project to explore the fundamentals of modern rendering pipelines, shader programming, texture mapping, and 3D transformations.

---

## ✨ Features

### Earth Rendering
- Procedurally generated sphere geometry
- High-resolution Earth texture mapping
- Real-time rotation animation
- GPU-based rendering using GLSL shaders

### Star Field Simulation
- Randomly generated 3D star positions
- Hundreds of rendered celestial objects
- Orbital motion around the scene
- Independent star rendering pipeline

### Camera System
- Keyboard navigation (W, A, S, D)
- Mouse-controlled camera targeting
- Real-time scene exploration

### Graphics Techniques
- Model / View / Projection transformations
- Texture mapping
- Depth testing
- Real-time lighting calculations
- Custom Vertex Shader
- Custom Fragment Shader
